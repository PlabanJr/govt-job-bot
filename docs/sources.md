# Central Government Sources

Phase 2 sources (official portals):
- UPSC Recruitment Notices (advertisement-wise)
- SSC Notices (legacy portal)
- Railway Recruitment Board Chandigarh Employment Notices
- India Post Vacancies

Add new sources by registering a parser in `apps/api/src/ingestion/sources` and adding it to `apps/api/src/ingestion/sources/index.ts`.
